# Архитектура SmartLight Hub

SmartLight Hub — центральное звено платформы. Он управляет Zigbee-сетью, обрабатывает события с датчиков и синхронизируется с облаком.

## Спецификация

- **Процессор:** ARM Cortex-A72 (BCM2711), 4 ядра
- **Память:** 2 GB LPDDR4
- **Хранилище:** 32 GB eMMC
- **Сеть:** Gigabit Ethernet, Wi-Fi 5 (802.11ac)
- **Радио:** Zigbee 3.0 координатор + Bluetooth 5.0

## Управление устройствами

Хаб поддерживает до 100 одновременных Zigbee-устройств и 50 Bluetooth-устройств. Каждое устройство регистрируется с уникальным идентификатором и типом.

**Типы устройств:**

- `light` — управляемая лампа
- `dimmer` — диммер
- `sensor_motion` — датчик движения
- `sensor_light` — датчик освещённости
- `switch` — выключатель

## Локальные сценарии

Правила автоматизации хранятся локально на хабе и выполняются даже при потере соединения с облаком. Формат правил — JSON:

```json
{
  "trigger": {
    "device": "sensor_motion_living",
    "event": "motion_detected"
  },
  "condition": {
    "sensor_light_living": "< 50"
  },
  "action": {
    "device": "light_living",
    "command": "on",
    "brightness": 80
  }
}
```
