# SmartLight IoT Platform

SmartLight — это платформа для управления интеллектуальным освещением в жилых и коммерческих помещениях. Позволяет централизованно контролировать освещение, собирать аналитику энергопотребления и автоматизировать сценарии на основе датчиков присутствия и уровня естественного света.

## Архитектура

Система состоит из трёх уровней:

1. **Устройства** — Wi-Fi/Bluetooth лампы, диммеры, датчики движения и освещённости
2. **Хаб** — локальный контроллер (Raspberry Pi), управляющий устройствами и ретранслирующий данные в облако
3. **Облачный backend** — API для управления, хранение метрик, AI-аналитика

## Основные компоненты

- **SmartLight Hub** — локальный контроллер на базе Linux
- **SmartLight API** — REST API для управления освещением
- **SmartLight Mobile App** — приложение для iOS и Android
- **SmartLight Dashboard** — веб-интерфейс аналитики

## Протоколы

- Устройства ↔ Хаб: Zigbee 3.0 / Bluetooth Mesh
- Хаб ↔ Облако: MQTT over TLS
- Клиенты ↔ API: HTTPS REST + WebSocket для real-time событий

## Преимущества

- Энергосбережение до 60% за счёт адаптивного освещения
- Поддержка сценариев "Умный дом" через IFTTT-подобные правила
- Открытый API для интеграции с第三方 системами
- Локальное управление без интернета
