## Findings
- Root cause of 11% ingest conversion: `max_completion_tokens=8000` truncated LLM JSON output, page silently dropped without escalation
- Wiki data is in user workspaces (`wiki-data/workspaces/<user_hash>/`), not in main `wiki-data/wiki/`
- Main `wiki-data/wiki/` is a template — 2 pages only
- ai@kulinich.ru workspace has 209 pages, all in `_general`
- Container restart invalidates session cookies — users need to re-login
- `docker compose down` removes `ai_personal_wiki_default` network, breaking traefik connectivity if prod file isn't used
